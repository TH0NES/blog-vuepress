# nrm

# 介绍

nrm(npm registry manager)是npm的镜像源管理工具，有时候国外资源太慢，使用这个就可以快速地在 npm源间切换

# 安装

npm install -g nrm
1

# 使用指定镜像源

nrm use cnpm
1

# 添加镜像源地址

nrm add <registry> <url>
1

registry为源名,url为镜像源地址

nrm add registry http://registry.npm.frp.trmap.cn/
1

# 删除镜像

nrm del <registry>
1
